
**Dependencies**
 - MongoDB
 - Rabbitmq

----

**Getting started**
```
  - npm install
  - npm install fuge -g
  - fuge shell fuge/fuge-dev.yml
  - fuge > start all
  ```

Now there is up some services:
  - API less-REST, localhost:9000/panel
  - Admin panel, localhost:8000
